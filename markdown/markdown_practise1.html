9. 插入视频

Markdown 语法是不支持直接插入视频的
普遍的做法是 插入 HTML 的 iframe 框架，可以通过网站自带的分享功能获取

# 代码如下
<iframe height=498 width=510 src='http://player.youku.com/embed/XMjgzNzM0NTYxNg==' frameborder=0 'allowfullscreen'></iframe>
