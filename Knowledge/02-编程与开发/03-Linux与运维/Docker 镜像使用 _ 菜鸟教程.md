2. 所以，我们如果要使用版本为15.10的ubuntu系统镜像来运行容器时，命令如下：

runoob@runoob:~$ docker run -t -i ubuntu:15.10 /bin/bash

-i: 交互式操作
3. ubuntu:15.10: 这是指用 ubuntu 15.10 版本镜像为基础来启动容器
