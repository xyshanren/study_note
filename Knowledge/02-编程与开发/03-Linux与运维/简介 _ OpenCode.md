# 简介 | OpenCode

> source: [opencode.ai](https://opencode.ai/docs/zh-cn/)

## 核心要点

1. OpenCode 是一个开源的 AI 编码代理。它提供终端界面、桌面应用和 IDE 扩展等多种使用方式
2. 前提条件

要在终端中使用 OpenCode，你需要：

一款现代终端模拟器，例如：

WezTerm，跨平台

Alacritty，跨平台

Ghostty，Linux 和 macOS

Kitty，Linux 和 macOS

你想使用的 LLM 提供商的 API 密钥
3. 配置

通过 OpenCode，你可以配置 API 密钥来使用任意 LLM 提供商
