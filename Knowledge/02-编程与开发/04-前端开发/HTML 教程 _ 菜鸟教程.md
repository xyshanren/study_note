# HTML 教程 | 菜鸟教程

> source: [www.runoob.com](https://www.runoob.com/html/html-tutorial.html)

## 核心要点

1. 您可以使用 HTML 来建立自己的 WEB 站点，HTML 运行在浏览器上，由浏览器来解析
2. 使用本站的编辑器，您可以轻松实现在线修改 HTML，并查看实例运行结果
3. 注意：对于中文网页需要使用 <meta charset="utf-8"> 声明编码，否则会出现乱码。有些浏览器(如 360 浏览器)会设置 GBK 为默认编码，则你需要设置为 <meta charset="gbk">
