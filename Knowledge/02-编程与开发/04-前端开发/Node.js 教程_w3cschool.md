## 核心要点

1. js对一些特殊用例进行了优化，提供了替代的API，使得V8在非浏览器环境下运行得更好。
2. js是一个基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。
3. js 使用事件驱动， 非阻塞I/O 模型而得以轻量和高效，非常适合在分布式设备上运行数据密集型的实时应用。
4. js 类 别 JavaScript工具 发布时间 2009年5月 开发者 Ryan Dah
5. 2009年2月，Ryan Dahl在博客上宣布准备基于V8创建一个轻量级的Web服务器并提供一套库。
