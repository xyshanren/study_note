# Docker 容器使用 | 菜鸟教程

> source: [www.runoob.com](https://www.runoob.com/docker/docker-container-usage.html)

## 核心要点

1. Docker 容器将应用程序及其所有依赖项（包括库、配置文件、系统工具等）封装在一个标准化的包中，使得应用能够在任何地方一致地运行
2. 容器是操作系统级别的虚拟化，不需要运行完整的操作系统，启动和运行更为高效
3. 镜像与容器的关系
镜像（Image）：容器的静态模板，包含了应用程序运行所需的所有依赖和文件。镜像是不可变的
