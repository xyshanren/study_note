SELECT INTO 语句从一个表复制数据，然后把数据插入到另一个新表中
3. SELECT EmployeeID, FirstName, LastName, Age, Department

INTO employees_backup

FROM employees

WHERE Age > 25;

使用注意事项

 表结构：

 SELECT INTO 会创建一个新表，并且新表的结构将基于选择的列和数据类型
