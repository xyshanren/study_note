 Python基础：数值（布尔型、整型、长整型、浮点型、复数）

一、概述

Python中的 数值类型（Numeric Types）共有5种：布尔型（bool）、整型（int）、长整型（long）、浮点型（float）和复数（complex）
2. 每一个Python对象都天生具有布尔值（True或False），进而可用于布尔测试（如用在if、while中）
3. >>> bool(A), bool(A())
(True, True)
>>> bool(B), bool(B())
(True, False)
>>> bool(C), bool(C())
(True, False)
