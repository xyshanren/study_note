Matplotlib 3.10.3 文档#

Matplotlib 是一个用于创建静态、动画和交互式可视化图表的综合性库。

安装#

pip
pip install matplotlib

conda
conda install -c conda-forge matplotlib

pixi
pixi add matplotlib

uv
uv add matplotlib

警告

如果您使用 uv 安装 Python，则 tkagg 后端将不可用，因为 python-build-standalone（uv 用于分发 Python）不包含 Matplotlib 可用的 tk 绑定（详情请参阅此问题）。如果您希望 Matplotlib 能够在窗口中显示图表，您应该安装其他支持的 GUI 框架之一，例如：

uv add matplotlib pyside6
