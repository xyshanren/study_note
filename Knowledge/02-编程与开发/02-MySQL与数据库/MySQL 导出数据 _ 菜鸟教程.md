# MySQL 导出数据 | 菜鸟教程

> source: [www.runoob.com](http://www.runoob.com/mysql/mysql-database-export.html)

## 核心要点

1. INTO OUTFILE 语句导出数据

SELECT...INTO OUTFILE 是 MySQL 中用于将查询结果导出到文件的语法
2. SELECT...INTO OUTFILE 允许你将查询的结果写入一个文本文件，基本的使用方法：

SELECT column1, column2, ..
3. INTO OUTFILE 'file_path'

FROM your_table

WHERE your_conditions;

参数说明：

column1, column2, ...: 要选择的列
