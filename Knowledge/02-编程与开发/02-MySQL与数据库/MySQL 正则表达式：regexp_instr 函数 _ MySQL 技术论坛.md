 在 MySQL 中， REGEXP_INSTR() 函数返回与正则表达式模式匹配的子字符串的起始索引
3. 语法

语法如下：

REGEXP_INSTR(expr, pat[, pos[, occurrence[, return_option[, match_type]]]])
expr 为源字符串，pat 为正则表达式
