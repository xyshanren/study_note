# Jenkins User Documentation

> source: [www.jenkins.io](https://www.jenkins.io/doc/)

## 核心要点

1. If you want to extend the functionality of Jenkins by developing your own
Jenkins plugins, please refer to the Extend Jenkins
(developer documentation)
2. Jenkins is a self-contained, open source automation server which can be used to
automate all sorts of tasks related to building, testing, and delivering or
deploying software
3. Jenkins can be installed through native system packages, Docker, or even run
standalone by any machine with a Java Runtime Environment (JRE) installed
