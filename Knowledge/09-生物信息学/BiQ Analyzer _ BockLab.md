BiQ Analyzer

Bisulfite sequencing analysis

Manual processing of DNA methylation data from bisulfite sequencing is a tedious and error-prone task. Here we present BiQ Analyzer, an interactive software tool that provides start-to-end support for this process. In an easy-to-use manner, the tool helps the user to import the sequence files from the sequencer, to align them, to exclude or correct critical sequences, to document the experiment, to perform basic statistics and to produce publication-quality diagrams.
