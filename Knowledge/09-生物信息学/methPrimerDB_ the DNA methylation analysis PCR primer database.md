# methPrimerDB: the DNA methylation analysis PCR primer database

> 来源：[methprimerdb.cmgg.be](https://methprimerdb.cmgg.be/)

## 核心要点

1. We strongly encourage researchers to submit their validated primer sequences
2. One can query the database using the official gene name, Entrez Gene or
methPrimerDB identifier, or oligonucleotide sequence
3. Different options make it possible to restrict a query to one organism (Human, Mouse, Rat) or method.
