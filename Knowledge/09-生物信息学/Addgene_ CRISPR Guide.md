# Addgene: CRISPR Guide

> source: [www.addgene.org](https://www.addgene.org/CRISPR/guide/)

## 核心要点

1. BLAST returns plasmids with similarity to the query sequence
2. Results are sorted by E-value, a statistic from BLAST that describes the significance of a match
3. Lower values are considered better matches
