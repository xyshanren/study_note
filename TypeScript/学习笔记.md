## TypeScript环境搭建
### 1. 安装 Node.js（必需） TypeScript 依赖 Node.js 运行
#### **Windows：**
```powershell
# 方法一：官网下载
# https://nodejs.org/ → 下载 LTS 版本（.msi 安装包，一路下一步）

# 方法二：命令行（推荐）
winget install OpenJS.NodeJS.LTS
```

**验证安装：**
```powershell
node --version
npm --version
```
### 2. 全局安装 TypeScript 编译器 
```powershell
npm install -g typescript
# 或局部安装
npm install --save-dev typescript
```
**验证：**
```powershell
tsc --version
```
 
 ### 3.安装代码编辑器（VS Code 推荐）

| 编辑器 | 下载 |
|--------|--------|
| VS Code（推荐） | https://code.visualstudio.com/ |
| WebStorm | https://www.jetbrains.com/webstorm/ |

**VS Code 必装插件：**
- **TypeScript Importer** — 自动导入类型
- **ESLint** — 代码风格检查
- **Prettier** — 代码格式化
### 4. 创建你的第一个 TypeScript 项目
```powershell
# 创建项目目录
mkdir my-ts-project
cd my-ts-project # 初始化 npm 项目
npm init -y # 安装 TypeScript（本地安装，项目专用）
npm install --save-dev typescript # 初始化 tsconfig.json（TypeScript 配置文件）
npx tsc --init
```
### 5. 目录结构建议 
```
my-ts-project/
├── src/ # 源代码目录
│ ├── index.ts # 入口文件
│ └── utils.ts # 工具函数
├── dist/ # 编译输出目录（编译后自动生成）
├── node_modules/ # 依赖
├── package.json
└── tsconfig.json # TypeScript 配置
```
### 6. tsconfig.json 核心配置（可直接用）
```json
{  
    "compilerOptions": {  
        "target": "ES2020",  
        "module": "commonjs",  
        "outDir": "./dist",  
        "rootDir": "./src",  
        "strict": true,  
        "esModuleInterop": true,  
        "skipLibCheck": true,  
        "forceConsistentCasingInFileNames": true,  
        "resolveJsonModule": true  
    },  
    "include": [  
        "src/**/*"  
    ],  
    "exclude": [  
        "node_modules",  
        "dist"  
    ]  
}
```
### 7. 常用命令
```powershell
# 编译整个项目
tsc # 监听文件变化，自动编译（开发时用）
tsc --watch # 编译单个文件
tsc src/index.ts # 运行编译后的 JS 文件
node dist/index.js
```
### 8. 一键启动命令（package.json scripts）
```json
{  
    "scripts": {  
        "build": "tsc",  
        "dev": "tsc --watch",  
        "start": "node dist/index.js"  
    }  
}
```
```powershell
npm run dev # 开发模式（热编译）
npm run build # 构建生产版本
```
